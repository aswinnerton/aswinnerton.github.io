Admin Console

Admin Command List:

CREATE: will create a vehicle for use, use in format (ignore parentheses): CREATE (reg) (brand)

Example: CREATE ABCD AUDI

OUTPUT: print the garage list, use in format: OUTPUT GARAGE
CHECKIN: check in an existing vehicle into the garage, use in format: CHECKIN (reg) (no. of faults)

Example: CHECKIN ABCD 2

CHECKOUT: check out an existing vehicle from the garage, use in formst: CHECKOUT (reg)
